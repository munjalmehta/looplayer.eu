DatenqualitätData Quality

Kein Datenmüll.
Vier Qualitätsstufen.No garbage data.
Four quality levels.

LoopLayer wendet das Datenstufen-Framework aus DIN SPEC 91446 — ursprünglich für Kunststoffrezyklate entwickelt, durch EN 18065 auf europäischer Ebene verankert — auf Textilfasern an. Jede Transaktion erhält eine Datenstufe (DQL 1–4). Compliance-Dokumente werden strukturell nur ab DQL 2 generiert. Das ist keine Policy — das ist eine Datenbankregel. LoopLayer applies the Data Quality Level framework from DIN SPEC 91446 — originally developed for plastic recyclates, anchored at European level through EN 18065 — to textile fibres. Every transaction is assigned a DQL level. Compliance documents are generated only from DQL 2 upward. This is not a policy — it is a database rule.
